Web Design

I believed that everyone has the same experience of web design in elementary school, however, we do not know what the meanings of your codes. In general, lots of people think that web design is a very difficult thing, in fact, that is not so difficult than you thought, except for amination and interface design. There is only one thing that you nee need to know - Practice as more as you can! I suggest that you may start from HTML and CSS, then you can learn more about its application and programming methods, including Javascript, JQuery, MySQL and PHP, etc.

Due to the limit of development time, this website only provides lots of links that I have used before. All contents are consist of resources for web design and examples. If you have interests in web design, you may develop your own website through Adobe Dreamweaver, Notepad++, Sublime, w3schools, and Brackets, etc. Please feel free to ask me, thank you very much!
